# 115 Proyecto 3 - Capturador de Productos con React

## Base de Datos de Productos

```yaml
tienda-db-service:
    # build: ./tienda-db
    build:
        context: ./tienda-db
        # dockerfile: Dockerfile
        target: test
        # target: prod
    environment:
        MYSQL_DATABASE: tienda
        MYSQL_USER: admin
        MYSQL_PASSWORD: tienda_admin_123
    networks:
        - tienda-net-db
```

## Api de captura de Productos

```yaml
tienda-api-service:
    depends_on:
        - tienda-db-service
    build: ./tienda-server
    environment:
        DB_HOST: tienda-db-service
        DB_PORT: 3306
        DB_NAME: tienda
        DB_USER: admin
        DB_PASSWORD: tienda_admin_123
    networks:
        - tienda-net-db
        - tienda-net-api
```

## Interfaz de captura de Productos

```yaml
tienda-app-service:
    depends_on:
        - tienda-api-service
    build: ./tienda-app
    networks:
        - tienda-net-api
        - tienda-net-app
```

## Proxy de captura de Productos

```yaml
tienda-proxy-service:
    depends_on:
        - tienda-db-service
        - tienda-api-service
        - tienda-app-service
    build: ./tienda-proxy
    networks:
        - tienda-net-api
        - tienda-net-app
```
